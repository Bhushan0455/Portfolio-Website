# Priyanshu Chauhan Portfolio

> A premium founder portfolio built with React, Vite, and Tailwind CSS, showcasing the journey, philosophy, and work of **Priyanshu Chauhan**, Founder & CEO of **Beyond Bound®**.

**Live Website:** https://www.priyanshuchauhan.com

---

# Overview

This project is a high-performance, SEO-optimized single-page application designed to represent the personal brand of Priyanshu Chauhan.

Unlike a traditional portfolio, the website serves as a living record of a founder's journey—from agriculture and healthcare strategy to building evidence-based metabolic health products through Beyond Bound®.

The application emphasizes:

* Performance
* Accessibility
* Technical SEO
* Modern UI/UX
* Responsive Design
* Smooth Motion Design
* Search Engine Discoverability

# Local Development

## Clone Repository

```bash
git clone <repository-url>

cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

Application runs at

```
http://localhost:5173
```

---

# Production Build

```bash
npm run build
```

Preview production build locally

```bash
npm run preview
```

---

# Deployment

The project is configured for deployment on **Vercel**.

Every push to the main branch automatically:

* Builds the application
* Generates static assets
* Runs React Snap pre-rendering
* Deploys to production
